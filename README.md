## Scripts

### npm run dev
- dev 모드 실행

### npm run build
- dist 폴더 에 빌드

### npm run zip
- 크롬 스토어 게시용 압축파일 생성
- dist 폴더 압축, 압축 된 파일 경로 `build/`
- **build 폴더 수동으로 1회 생성 필요!**

### npm run docs
- typedoc 문서 생성 경로 `docs/`
